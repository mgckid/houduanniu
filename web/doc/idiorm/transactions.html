Transactions露

Idiorm doesnt supply any extra methods to deal with transactions, but its very easy to use PDOs built-in methods:

<?php
// Start a transaction
ORM::get_db()->beginTransaction();

// Commit a transaction
ORM::get_db()->commit();

// Roll back a transaction
ORM::get_db()->rollBack();

For more details, see the PDO documentation on Transactions.
